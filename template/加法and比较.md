## 公式

``` 
当前位 = (A 的当前位 + B 的当前位 + 进位carry) % 10
```

## 加法模板

``` 
while ( A 没完 || B 没完 || carry > 0)
    A 的当前位
    B 的当前位

    和 = A 的当前位 + B 的当前位 + 进位carry

    当前位 = 和 % 10;
    进位 = 和 / 10;
    添加进数组
    
数组反转
返回数组

例子： 2.两数相加、989.数组形式的整数加法
```



## 字符比较模板

``` 
while( A 没完 && B 没完)
    A 的当前字符
    B 的当前字符

    A 的当前字符长度
    B 的当前字符长度

    判读符合比较条件吗
判断 A B 都走完了吗

例子： 809.情感丰富的文字
```
